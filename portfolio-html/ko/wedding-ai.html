데이터 기반 업체 추천

막연한 웨딩 플래닝 결정을 구조화된 추천 흐름으로 바꾸는 시스템.

문제

  • 플랫폼 간 파편화된 정보
  • 들쭉날쭉한 가격, 어려운 비교
  • 업체 선택이 느리고 스트레스적이며 불투명

솔루션

  • RAG 기반 추천 아키텍처
  • 자연어 → 구조화된 검색 조건
  • 높은 마찰과 많은 선택이 있는 의사결정을 위해 설계

미리보기

기여

  • 업체 데이터 전반의 스키마 및 임베딩 파이프라인 설계
  • 자연어 쿼리에서 필터 추출 구현
  • AI 지원 의사결정 도구로 포지셔닝

기술 스택

Supabase PostgreSQL pgvector Python OpenAI API

비즈니스 임팩트

  • 고마찰 마켓플레이스 결정에서 AI의 역할 실증
  • 확신이 필요한 복잡한 선택 상황에서 명확성 제공
  • 카테고리 기반 추천 모델로 확장 가능
