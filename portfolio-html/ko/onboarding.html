HR 특화 AI 어시스턴트 · B2B SaaS / HR / AI

파편화된 문서, 반복 구두 안내, 첫 주 정보 과부하를 없애기 위해 설계된 모바일 중심 온보딩 어시스턴트입니다.

문제

신입사원은 여러 문서에 흩어진 정보와 반복 구두 설명을 받습니다 — 직원에게는 마찰을, HR팀에는 반복적인 수동 업무 부담을 줍니다.

솔루션

FAQ 카드, 체크리스트 흐름, AI 챗봇 지원을 하나의 온보딩 경험으로 통합하며, AI 실패 시에도 신뢰성을 위한 폴백 로직을 포함했습니다.

  • FAQ, 체크리스트, 챗봇을 하나의 통합 흐름으로
  • 가상 사례가 아닌 실제 HR 운영 기반으로 설계
  • AI 실패 시나리오를 위한 폴백 UX 설계

기여

  • 실제 온보딩 운영에서 전체 사용자 흐름 정의
  • 챗봇 연동 및 폴백 인터랙션 로직 구현
  • 명확한 B2B SaaS 활용 사례로 포지셔닝

기술 스택

React Native (Expo) Node.js Express OpenAI API

비즈니스 임팩트

AI가 HR 기업 워크플로우에서 운영 마찰을 줄이고 측정 가능한 가치를 창출하는 방식을 보여줍니다.
