Onboarding
Assistant

HR-focused AI Assistant · B2B SaaS / HR / AI

A mobile-first onboarding assistant built to eliminate the friction of fragmented documents, repeated verbal guidance, and first-week information overload.

The Problem

New hires get information scattered across documents and repeated verbal guidance — creating friction for employees and recurring manual burden for HR teams.

The Solution

Combines FAQ cards, checklist flows, and AI chatbot support in a single onboarding experience — with fallback logic for reliability when AI responses fail.

  • FAQ, checklist, chatbot in one unified flow
  • Built from real HR operations, not hypothetical cases
  • Fallback UX designed for AI failure scenarios

My Contribution

  • Defined full user flow from real onboarding operations
  • Built chatbot integration and fallback interaction logic
  • Positioned product as a clear B2B SaaS use case

Tech Stack

React Native (Expo) Node.js Express OpenAI API

Business Impact

Shows how AI reduces operational friction and creates measurable value in HR-facing enterprise workflows.
