Why This Matters

The complexity problem no spreadsheet solves

Shift-based pay is the most rule-laden, least standardized area of payroll.

01 — The Complexity Problem

Overlapping rules make manual calculation unreliable

Night premiums, holiday multipliers, overtime thresholds — each workplace applies these differently and in combination. A spreadsheet handles one set of rules. It breaks the moment a worker changes jobs or their employer updates the pay structure.

02 — The Trust Gap

Workers don't just want an answer — they want to understand it

A black-box "your pay is ₩2,340,000" doesn't build trust. Workers need to see the breakdown: base hours, night premium applied, holiday shifts counted. The product had to be transparent and auditable to be actually used — not just technically correct.

⚡

The real opportunity: translate the most frustrating monthly calculation into a reliable, transparent, explainable product — with a clear monetization path through premium feature gating built in from day one.

Demo Scenario

What a client sees in 3 minutes

From shift input to monthly report to premium upsell — a complete, demoable product loop.

1
Setup

User selects or creates a workplace profile

Profile stores the custom rule set: base hourly rate, night shift premium (e.g. 1.5×), holiday multiplier (e.g. 2×), and overtime threshold. One-time setup, reused every month.

2
Shift Input

23 night shifts, 3 holiday shifts, 8 OT hours entered

User taps in their monthly shift log. Live calculation preview updates in real time as each shift is entered — no waiting, no "calculate" button.

3
Rule Engine

Rule engine applies all combinations simultaneously

Base rate × night premium × holiday multiplier × OT threshold — all applied correctly and simultaneously. The calculation handles overlapping rules that break spreadsheets.

4
Monthly Report

Report generated: total pay + breakdown by category

User sees total pay with a transparent breakdown: base pay, night premium subtotal, holiday pay, OT amount. Every line is auditable. Trust built from visibility, not just accuracy.

5
Premium Gate

Export to PDF triggers in-app purchase flow

User taps "Export Report." Premium upgrade prompt appears — natural trigger, high intent moment. In-app purchase unlocks PDF export, multi-month history, and analytics. Monetization built into the product loop.
