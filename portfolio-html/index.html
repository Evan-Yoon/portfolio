Accessibility Solution Proposal

WalkMate는 시각장애인의 보행 문제를
기술 기반 안전 솔루션으로 재구성한 프로젝트입니다.

실시간 위험 인지, 안전한 음성 안내, 현장 데이터 축적을 하나의 흐름으로 연결해 PoC 수준의 end-to-end 제안으로 구현했습니다.

Project Overview

문제는 길찾기 부족이 아니라, 전방 위험을 설명해주는 안내 공백이었습니다.

기존 보조도구와 일반 내비게이션은 발밑 또는 경로 중심이라 실제 보행 중 마주치는 장애물, GPS 흔들림, 안내 신뢰성 문제를 충분히 해결하지 못했습니다.

  • 사용자 관점에서는 안전한 이동 경험의 부재
  • 운영 관점에서는 위험 데이터를 축적할 구조의 부재

Solution Architecture

온디바이스 AI, Tmap, 음성 UX, 관리자 대시보드를 한 흐름으로 묶었습니다.

앱에서 위험을 감지하고, 즉시 음성으로 행동을 안내하며, 동일한 이벤트를 운영 데이터로 전환해 관리 화면까지 이어지도록 설계했습니다.

  • 지연을 줄이기 위한 온디바이스 추론 기반 위험 감지
  • Tmap 경로와 방향 안내를 결합한 음성 중심 UX
  • 위험 이미지·좌표를 축적하는 관리자 대시보드 연동

My Role & Demo

저는 팀장 겸 프론트엔드 담당으로, “어떻게 보여줄 것인가”를 설계했습니다.

  • 사용자 접점이 되는 모바일 UX와 데모 시나리오를 직접 설계
  • 모바일, AI, 백엔드, 관리자 화면을 연결한 end-to-end PoC 주도
  • 이해관계자별로 다른 가치 제안과 설명 포인트를 구성

Technical Decisions

트러블슈팅은 기능 수정이 아니라, 신뢰 가능한 데모와 운영성을 만드는 작업이었습니다.

  • TTS 중복은 우선순위 큐로 정리해 사용자 피로도와 혼선을 줄였습니다.
  • GPS 흔들림은 방향 안정화 로직으로 보정해 안내 신뢰성을 높였습니다.
  • API 구조와 S3 권한 이슈를 정리해 연동 안정성과 데모 완성도를 높였습니다.
Team 윤지현 UI/UX · Team Lead 이지호 Back-End 최현석 Model

Business Value

같은 기술도 고객마다 다른 가치로 설명할 수 있어야 실제 제안이 됩니다.

WalkMate를 단순 앱이 아닌 B2G/B2B형 보행 안전 솔루션으로 포지셔닝하며 TAM/SAM/SOM, 수익 구조, 파트너십, ROI까지 함께 검토했습니다. 사용자에게는 안전한 이동 경험을, 지자체에는 위험구역 관리 데이터를, PM 업체에는 운영 효율을, ESG 기업에는 측정 가능한 사회적 임팩트를 제안했습니다.
